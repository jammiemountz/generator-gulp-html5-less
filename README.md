## html5-less with Gulpjs Generator

### Getting Started

To install generator-html5less from npm, run:

```bash
npm install -g generator-html5less
```

Finally, initiate the generator:

```bash
yo html5less
```

To run Gulpjs type in terminal:
```bash
gulp
```
